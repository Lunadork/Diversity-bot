Hazibot API

By Henry, Arvin, Zeia and Ikenna


How to use


To use send a JSON POST request to /bot


Sending requests


Your request body needs to contain the following:

{ "username" : "user_username", "message" : "the message content", "context" : [context] }


Username and message should be strings.


Context


Context is an array containing strings.

It is how Hazibot tracks where in a conversation they may be.

Send an array with one string value -> "none" for context in your initial message, i.e. "context" : ["none"]


Hazibot's returns


Hazibot will pass back a JSON object constructed as such:

{ "message" : "message content", "context" : [context], "expression" : 1 }

You should return the context Hazibot gives with any followup message your user also sends


Expression


Expression means the tone of Hazibot's reply

It will be returned as an integer that you can use on your frontend


  1. Means neutral
  2. Means happy
  3. Means sad
  4. Means pensive

Hazibot will store data the user passes in and recall based on username, so it must be unique.

There is currently no way to register users via a form and can only be done by speaking with Hazi via the Diversity MH app
